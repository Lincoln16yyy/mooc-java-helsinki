# Java Programming - University of Helsinki (MOOC.fi)

This repository contains my solutions to the world-renowned Java Programming course from the **University of Helsinki**.

## 🚀 Key Highlights
* **Validated Learning:** Every exercise has been verified by the **Test My Code (TMC)** automated testing system, ensuring logical correctness and code quality.
* **Comprehensive Scope:** Covers everything from basic syntax to advanced **Object-Oriented Programming (OOP)**, including interfaces, polymorphism, and stream APIs.
* **Professional Environment:** Developed on **Ubuntu Linux** using **VS Code** and managed with **Git**.

## 🛠️ Technologies & Skills
* **Language:** Java
* **Version Control:** Git & GitHub
* **Environment:** Linux (Ubuntu)
* **Methodology:** Test-Driven Development (TDD) principles through TMC
